xyzzy の backward-word と previous-word, forward-word と next-word の違い

xyzzy の lisp の関数 backward-word, previous-word, forward-word, next-word の動作の違いがよくわからなかったので、試してみた。

黒背景白文字が移動前のカーソルの位置で、灰色背景太字が移動後の場所。

backward-word

123　456　789

123　456　789

123　456　789

previous-word

123　456　789

123　456　789

123　456　789

forward-word

123　456　789

123　456　789

123　456　789

next-word

123　456　789

123　456　789

123　456　789

backward-word と previous-word は同じ動作で forward-word と next-word は動作が異なるらしい。 あってんのかな、これ。
