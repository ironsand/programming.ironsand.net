Net::HTTPを使ってリモートにあるサーバーの最終更新日を取得

更新されてる時だけファイルをダウンロードしたいときなどに使う目的で。

uri = URI('http://example.com/foo.xls')
Net::HTTP.get_response(uri)['last-modified']
"Mon, 08 May 2017 01:34:46 GMT"
