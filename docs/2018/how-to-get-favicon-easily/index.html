Faviconを取得する簡単な方法

faviconを取得したければ<link rel="icon" href="favicon.ico">を探すなどの正攻法がありますが、

色々と例外があるようなのでめんどくさい時はGoogleさんに任せてしまいましょう。

URI('https://www.google.com/s2/favicons?domain_url=stackoverflow.com').read

でとれます。
