gem `mail` と sendmail を使って簡単にRubyでメール送信

net/smtp ライブラリを使うよりも gem mail を使えば簡単にメールが送れるらしいので調べてみた。

方法

まず mail gem が必要なので $ gem install mail でインストールする。

んで、下のように書けばOK!

require 'mail'
mail = Mail.new do
  from     'me@ironsand.net'
  to       'recipient@exmaple.com'
  subject  'Here is the image you wanted'
  body     'Body'
end

mail.delivery_method :sendmail

mail.deliver

注意点

Macでは動かなかったので、VPS上のCentOS 6.4で試してます。

25版ポートを開けるのを忘れずに。
